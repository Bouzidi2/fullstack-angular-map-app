# Full-Stack Angular Map Application

## 🚀 Overview
A full-stack web application built using Angular and Spring Boot that displays dynamic country data through an interactive map interface. The application integrates frontend event-driven components with backend REST APIs to retrieve and display real-time information.

This project demonstrates full-stack development using modern Java frameworks, frontend frameworks, and API-driven architecture.

## 🛠 Tech Stack
- Java
- Spring Boot
- Angular
- TypeScript
- REST APIs
- HTML / CSS
- Maven

## ⚙️ Features
- Interactive world map UI built with Angular
- Dynamic data retrieval using HTTP requests
- Event-driven components for user interaction
- Integration with backend REST APIs
- Responsive UI with real-time updates

## 📦 Architecture
- Angular frontend handles UI rendering and user interaction
- REST API provides data for country information
- Event-driven communication between components
- Modular structure for scalability and maintainability

## ▶️ How to Run
1. Clone the repository
2. Run the Spring Boot backend application
3. Run the Angular frontend application
4. Open the application in a browser and interact with the map

## 📌 Key Takeaways
- Built a full-stack application using Angular and Spring Boot
- Implemented API integration with dynamic data rendering
- Developed event-driven frontend components
- Applied full-stack development principles across multiple technologies
